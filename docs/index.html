KDMM Kodi Repository

KDMM (Kodi Debrid Media Manager) plays TMDb Bingie Helper movie and episode requests through Debrid Media Manager search results and cached Real-Debrid or AllDebrid streams.

It resolves direct playback URLs, ranks candidates by quality preferences, filters AV1 streams, retries fallback streams when playback fails to start, caches stream and TV pack choices, saves resume progress, and provides IntroDB-backed skip/next-episode overlays.

Install

Download and install the repository zip in Kodi: repository.unrefundable.kdmm-1.0.0.zip
